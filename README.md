# 停车场管理系统 java1071

本项目是基于Spring Boot和Vue的前后端分离停车场管理系统。

## 技术栈

- 开发工具：IDEA
- 运行环境：JDK 1.8 + Maven + MySQL 8 + npm
- 技术框架：Java + Spring Boot + MySQL + MyBatis + Maven + Vue

## 功能模块

- 登录
- 注册
- 后台首页
- 用户信息管理
- 车辆信息管理
- 新增车辆
- 车位费用设置
- 停泊车辆查询
- 车辆进出管理
- 登录日志查询
- 个人中心
- 预定停车位
- 缴费信息

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 服务器：推荐使用Linux系统
- 数据库：MySQL 8及以上版本
- 前端：现代浏览器

## 部署步骤

1. 准备运行环境：JDK 1.8、Maven、MySQL、npm等
2. 导入项目数据库
3. 编译后端项目，并部署到服务器
4. 编译前端项目，并部署到Web服务器

## 目录结构

```
- backend  # 后端代码目录
- frontend # 前端代码目录
- database # 数据库文件目录
```

## 核心亮点

- 前后端分离架构，易于维护和扩展
- 完善的车辆管理和停车缴费功能
- 系统稳定性高，经严格测试
